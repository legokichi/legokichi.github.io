cuttleboneを使ってnar配布ページにデモゴを仕込む

この記事は 伺か Advent Calendar 2015 の12/24の記事です。

ゴースト開発者向けの記事です。

何これ

Webゴースト用シェル描画ライブラリ cuttlebone と
SakuraScript実行ライブラリ SakuraScriptPlayer.js と
Nar読み込みライブラリ NarLoader を利用して
Webゴーストの配布ページでゴーストのデモを動かそうという記事です。

つかいかた

  1. sample.html の <script>var NAR_URL =... から ssp.play(shioriRequest({ID:"OnBoot"}));});</script> までをコピる
  2. nar配布ページに貼り付ける
  3. var NAR_URL = "*.nar"; を配布しているnarへのURLにする
  4. SakuraScriptを書き換える(SSPのスクリプトログからコピペする)

注意事項

  • SSPのスクリプトログからコピペする場合 \ を \\ に置換するようにしましょう
