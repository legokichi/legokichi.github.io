Dockerfile内で libnvcuvid.so がない

libnvcuvid.so はドライバ付属ライブラリである。
プロビジョニング時にはまだドライバが配置されていない。

cp /usr/lib/nvidia-375/libnvcuvid.so ./
sudo  nvidia-docker build -t foo ./

COPY libnvcuvid.so /usr/local/lib

などするとよいかも
